什么是字节码

java bytecode 由单字节（byte）的指令组成，理论上最多支持256个操作码，实际只使用了200左右，其他的保留给调试操作

分类

  • 栈操作指令，包括与局部变量交互的指令
  • 流程控制指令
  • 对象操作指令，包括调用方法
  • 算数运算、类型转化指令

最简单的字节码

javac编译

javap -c 查看字节码

-c：分解方法代码，即显示每个方法具体的字节码

-public | protected | package | private：用于指定显示哪种级别的类成员

-verbose：指定显示更进一步的详细信息

指令：javap -c -verbose HelloByteCode.class
